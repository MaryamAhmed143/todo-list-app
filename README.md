# To-Do List App

A responsive To-Do List application built using HTML, CSS, and JavaScript.

# Features

- Add new tasks
- Add subtasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Responsive design
- Easy-to-use interface

# Technologies Used
- HTML5
- CSS3
- JavaScript
