# Task Manager Dashboard

A modern and responsive Task Manager Dashboard built using HTML, CSS, and JavaScript.

## Features

- ✅ Add, edit, and delete tasks
- 🎯 Priority levels
- 📂 Categories
- 📅 Due dates
- 📌 Pin important tasks
- 📋 Subtasks
- 🔍 Search tasks
- 📊 Progress tracker
- 🖱️ Drag and drop task ordering
- 💾 Local Storage support
- 📱 Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript
